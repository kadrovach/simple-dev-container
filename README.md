# Image Search

A web application that allows users to search for images using Unsplash API with authentication and containerized deployment

## Getting Started

### Prerequisites

- Node.js 18+
- Docker (optional, for containerized deployment)

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Test

```bash
npm test
```

### Docker

Build and run with Docker:

```bash
docker-compose up --build
```

The application will be available at http://localhost:4000
